Projeto: Sistema de Controle Financeiro Pessoal

Descrição

Este sistema permite o cadastro de entradas e saídas financeiras, categorização de gastos e definição de limites mensais.

Funcionalidades previstas

  • Cadastro de entradas (salário, freelas, etc)
  • Cadastro de saídas (boletos, cartões, compras)
  • Classificação por categorias
  • Definição de limites de gastos

Perguntas:

1) Como o TypeScript ajudaria a garantir o uso correto de req e res?

O TypeScript permitiria definir tipos para os objetos req e res, evitando erros comuns em tempo de execução. Assim, o editor e o compilador apontariam usos incorretos, como falta de métodos ou propriedades.

2) Como ficaria esse código em TypeScript? Benefícios e dificuldades?

Em TypeScript, o código usaria tipos explícitos para os objetos Request e Response, vindos do pacote @types/express. Os benefícios seriam maior segurança, autocompletar e detecção de erros na hora de programar. Como dificuldade, seria necessário configurar o TypeScript (tsconfig.json) e fazer o transpile para JavaScript antes de executar.
